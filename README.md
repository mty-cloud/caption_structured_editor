# 机标结构化修改器

> 把机标文本拆成「镜头 → 说话单元 → 字段」模块，结构化修改后自动生成标准文本。直连 Label Studio API，支持拉取/提交/自动下一任务。

适用于 **Windows / macOS** 双平台，无需安装任何环境，双击或 `python3 -m http.server` 即可使用。

---

## 🚀 快速使用

### 在线版（无需下载）

**https://mty-cloud.github.io/caption_structured_editor/**

### 本地运行

```bash
# 克隆或下载本项目
cd caption_structured_editor_mvp

# 启动 HTTP 服务器（任选其一）
python3 -m http.server 8000
# 或
npx serve .
```

浏览器打开 **http://localhost:8000**

## 📖 使用说明

### 基本流程

1. **⚙️ 配置** — 点击 ⚙️，确认 LS 服务器地址和项目 ID（首次使用需配置，自动保存到 `localStorage`）
2. **📥 从LS拉取** — 自动获取当前项目中第一个未标注任务，解析到右侧编辑器
3. **修改** — 在右侧卡片中逐字段修改说话内容、时间、说话人、性别、人物可见
4. **✅ 提交并下一个** — 自动提交标注并加载下一未完成任务
5. **🗑 丢弃此条** — 如果当前片段无需修改，丢丢弃后自动跳下一任务

### 编辑功能

| 功能 | 说明 |
|------|------|
| 解析机标文本 | 自动识别 `[镜头 N][start-end]` 和 `{...}` 说话单元，兼容非严格 JSON |
| 修改镜头时间 | 直接编辑镜头的开始/结束时间，相邻镜头自动联动 |
| 修改说话内容 | 修改每个说话单元的内容、时间、说话人、性别、人物可见 |
| 快捷选择 | 鼠标 hover 或点击字段 chip → 弹出选项面板直接选值 |
| 新增/删除镜头 | 灵活增删镜头，支持时间合并到相邻镜头 |
| 新增/复制/删除说话单元 | 支持复制、删除、上移、下移 |
| 校验 | 自动校验时间、字段格式、空内容、时间范围，实时显示 |

### 核心智能特性

| 特性 | 说明 |
|------|------|
| 🔄 说话人自动重编号 | 任何 speaker 修改后，自动按首次出现顺序重新分配 `speaker_0/1/2...` |
| 👫 性别自动同步 | 修改某一说话人的性别后，所有同 speaker 的性别一步到位同步更新 |
| 🚫 提交前风险控制 | 人物可见=「不确定」自动标记红色错误，禁止提交 |
| 🔴 不确定字段高亮 | 「不确定」字段显示为红色浮动脉冲 chip，一眼可定位 |
| 🗑️ 丢弃关键词检测 | 检测 memo 中的「分屏/渐变/定格/黑幕/丢弃/慢动作/快进」等关键词 |
| 🚨 全页红色横幅 | 检测到丢弃关键词时，页面顶部弹出闪烁红色横幅警告 |
| ⚙️ 弃关键词管理 | 可自定义检测的关键词列表，持久化保存 |

## 🧠 智能逻辑详解

### 说话人自动重编号

所有 `speaker_N` 的编号永远代表该说话人按**首次出现顺序**的序号。

**示例**：
```
修改前：speaker_0, speaker_0, speaker_5, speaker_1
重编号：speaker_0, speaker_0, speaker_1, speaker_2
```

每次修改说话人、增删或移动说话单元后自动触发，无需手动调整。

### 性别自动同步

修改某个说话人的性别时，所有同 speaker 的性别实时同步更新。

```
speaker_0 在镜头1=男，镜头2=男，镜头3=不确定
→ 你把镜头1的 speaker_0 性别改为「女」
→ 系统自动把所有 speaker_0 的性别更新为「女」
→ 一次修改，全自动化
```

### 提交前风险控制

点击「提交并下一个」时自动扫描所有说话单元：

| 风险类型 | 级别 | 处理方式 |
|----------|------|----------|
| 🔴 性别为空 或 不确定 | block | `alert` 禁止提交，无法绕过 |
| 🟡 人物可见为「不确定」 | block | `alert` 禁止提交，无法绕过 |
| 🟡 说话人为空 | warn | `confirm` 二次确认 |

### 丢弃关键词检测

从 LS 拉取任务后自动扫描 memo（上一轮备注）字段：

- 匹配关键词时 →
  - 页面顶部弹出**红色浮动横幅**（🚨 上一轮提示丢弃: xxx）
  - 任务栏同时显示红色 chip 提示
  - 点击横幅可关闭
- 关键词可自定义管理（「🗑️丢弃提示」按钮）
- 默认关键词：分屏、渐变、定格、黑幕、丢弃、慢动作、快进

## 🔌 Label Studio 集成

通过 REST API 直连 Label Studio，无需 DOM 注入或额外桥接层。

### 连接配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| LS 服务器地址 | `http://115.191.32.74:30014` | 通过 ⚙️ 设置面板修改 |
| 项目 ID | `4296` | 通过 ⚙️ 设置面板修改 |
| API Token | 固定预设值 | 只读，不可修改 |

设置项自动保存在浏览器 `localStorage`，刷新/重启后保留。

### 提交内容

| LS 字段 | 提交值 |
|---------|--------|
| `caption_v5_translate_update` | 修改后的完整机标文本 |
| `choice0` | `是`（丢弃）/ `否`（正常提交） |

### 提交循环

```
loadNextTask → 结构化修改 → submitAndNext → loadNextTask → ...
```

- 纯前端，**零依赖**，无需 Node.js / npm / Python 等运行环境
