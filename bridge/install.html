🔗 安装 LS 桥接书签

安装后，在任何 LS 项目页点击书签，即可把机标数据发送到编辑器。

📌 第一步：拖拽到书签栏

将下方按钮拖拽到浏览器的书签栏（收藏夹栏）中：

📌 LS桥接

💡 如果拖拽无效，可以右键点击上方按钮 → 「复制链接地址」→ 新建书签，粘贴到 URL 栏。

🔄 第二步：启动编辑器服务器

在项目目录执行：

python3 server.py

启动后打开 http://localhost:8000

📋 第三步：使用流程

  1. 在浏览器打开 LS 标注页面（如 http://115.191.32.74:30014/projects/4296/data?tab=3941）
  2. 点击书签栏的 「LS桥接」
  3. 首次使用会弹出对话框要求输入 API Token（只需输入一次，自动保存）
  4. 提示「桥接成功」后，切换到编辑器页面
  5. 点击 「📥 从LS桥接」 按钮
  6. 数据自动解析到右侧编辑器，开始修改

🔐 Token 说明

API Token 可在 LS 页面右上角头像 → Account → Access Token 找到。

Token 只会保存在你浏览器的 localStorage 中，不会上传到任何地方。
